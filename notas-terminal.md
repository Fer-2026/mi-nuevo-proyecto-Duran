# Comandos de terminal utilizadas

1. Comando pwd:
Que hace: Directorio actual.
Cuando lo use: para verificar en que carpeta estabab antes de crear el proyecto.

2. Comando ls:
Que hace: lista archivos.
Cuando lo use: para ver los archivos creados en mi proyecto.

3. Comando cd carpeta:
Que hace: entrar a carpeta.
Cuando lo use: lo use para entrar a la carpeta del proyecto.

4. Comando mkdir nombre:
Que hace: crear carpeta.
Cuando lo use: lo use para crear la carpeta de trabajo practico.

5. Comando touch archivo:
Que hace: crear archivo.
Cuando lo use: para crear el archivo principal del sitio web.

6. Comando cat archivo:
Que hace: ver contenido.
Cuando lo use: lo use para ver el contenido del archivo HTML.

7. Comando which programa:
Que hace: donde esta instalado Git.
Cuando lo use: lo use para verificar donde esta instalado Git en mi computadora.

8. Comando code:
Que hace: abrir Visual Studio Code aqui.
Cuando lo use: lo use para abrir mi proyecto en VS  Code.

9. Comando clear:
Que hace: limpiar pantalla.
Cuando lo use: lo use para limpiar la terminal mientras trabajaba.

10. Comando git --version:
Que hace: muestra la version instalada de Git.
Cuando lo use: lo use para verificar que Git estaba correctamente instalado.
